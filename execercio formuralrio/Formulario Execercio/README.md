# Sistema de Notas de Alunos

Sistema web para gerenciamento de notas escolares desenvolvido com Node.js e Express.

##  Funcionalidades

- Cadastro de alunos com nome, nota do 1º bimestre e nota do 2º bimestre
- Cálculo automático da média
- Definição da situação do aluno:
  -  **Aprovado**: média ≥ 6
  -  **Exame Final**: média ≥ 2 e < 6
  -  **Reprovado**: média < 2
- Tabela com todos os alunos cadastrados
- Persistência de dados via localStorage



Acesse: [http://localhost:3000](http://localhost:3000)
